# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Examples of behavior that contributes to a positive environment:
- being respectful and direct
- giving actionable technical feedback
- assuming good intent while challenging weak evidence
- keeping discussions focused on the code, docs, or operations impact

Examples of unacceptable behavior:
- harassment or personal attacks
- discriminatory or hateful language
- doxxing or publishing private information
- disruptive bad-faith engagement

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with it.

## Reporting

Please report conduct issues privately to the maintainers.
